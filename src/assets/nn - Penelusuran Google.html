      Untuk membuat gambar di popup berbeda-beda sesuai dengan data, Anda perlu memodifikasi bagian properti
      capabilities dengan menambahkan URL gambar spesifik untuk setiap kategori, dan memperbaiki sintaks pemanggilan variabel di bagian <img>. 
      Berikut adalah langkah-langkah perbaikannya: 
      1. Perbarui Data capabilities 
      Tambahkan properti image pada setiap objek. Anda bisa menggunakan layanan seperti Unsplash untuk gambar asli atau tetap menggunakan placeholder dengan teks yang berbeda. 
      javascript
      const capabilities = [
  { 
    icon: '', 
    title: 'Pengenalan Pola Kompleks', 
    short: 'Mengenali wajah, suara, & bahasa.', 
    description: 'Mampu mengenali pola yang tidak terstruktur...',
    // Gunakan URL gambar yang relevan
    image: 'https://images.unsplash.com' 
  },
  { 
    icon: '', 
    title: 'Pembelajaran Otomatis', 
    short: 'Belajar tanpa program manual.', 
    description: 'Sistem belajar dan beradaptasi secara otomatis...',
    image: 'https://images.unsplash.com'
  },
  // ... tambahkan image untuk yang lainnya
];
      Gunakan kode dengan hati-hati.
      2. Perbaiki Sintaks HTML (Template) 
      Kesalahan pada kode Anda sebelumnya adalah penggunaan tanda kurung kurawal {} di dalam atribut :src. Di Vue, jika sudah menggunakan titik dua (:), Anda cukup memanggil nama variabelnya. 
      Ubah bagian ini: 
      html
      <!-- SALAH -->
<img :src="`https://placehold.co{activeData.title}`" />

<!-- BENAR (Menggunakan properti image dari data) -->
<img :src="activeData.image" :alt="activeData.title" />
      Gunakan kode dengan hati-hati.
      Kode Lengkap Bagian Script (Opsi Placeholder Dinamis) 
      Jika Anda tetap ingin menggunakan placehold.co namun otomatis berubah sesuai judul, gunakan Template Literals yang benar dengan simbol dollar ${}: 
      html
      <div class="m-image-box">
  <!-- Pastikan menggunakan backtick (`) dan ${} untuk variabel -->
  <img :src="`https://placehold.co{activeData.title}`" />
</div>
      Gunakan kode dengan hati-hati.
      Tips Tambahan untuk Visual 2026: 
      Agar tampilan gambar di popup lebih profesional dan tidak kosong saat dimuat: 
      1. Loading State: Tambahkan atribut loading="lazy" pada tag img.
      2. Object Fit: Di bagian CSS, tambahkan object-fit: cover agar gambar tidak gepeng. 
      css
      .m-image-box img {
  width: 100%;
  height: 300px; /* Tentukan tinggi tetap */
  object-fit: cover; /* Gambar akan terpotong rapi, tidak ditarik */
  border-radius: 20px;
  border: 2px solid #334155;
}

              Turn on your Visual Search History?

              Google uses its visual recognition technologies to process the images you use to search, like when you search with Google Lens. If you turn on your Visual Search History, Google will save these images from eligible Google services to your Web & App Activity when youre signed in to your Google Account. You can learn more about this setting and which Google services save images to it at g.co/Search/VisualSearchHistory.

              How visual search history is used

              Your Visual Search History may be used to improve your experience on Google services, like letting you revisit your past visual searches. It may be used to develop and improve Googles visual recognition and search technologies, as well as the Google services that use them.

              When visual search history is off

              If you turn this setting off, any previous Visual Search History may still be kept and used to improve Googles visual recognition and search technologies, unless you delete it from your Web & App Activity.

              Visual Search History doesnt affect images saved by other settings, like Gemini Apps Activity.

              How to manage your Visual Search History

              You can view, delete, or manage your Visual Search History at activity.google.com. To download your Visual Search History, visit takeout.google.com. Images will be deleted in accordance with your Web & App Activity auto-delete settings, although some types of images may be deleted sooner.

              Google uses and saves data in accordance with Google Privacy Policy.
